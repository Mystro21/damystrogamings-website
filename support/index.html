Support for Da Mystro Gamings LLC

Da Mystro Gamings LLC is an emerging game studio based in Silver Spring, Maryland. For current support, please contact us by email.

Email Support

For help, questions, testing feedback, or issue reports, email Da Mystro Gamings LLC at:

damystrogamings21@gmail.com

What to include

Please include the game name, your device type, and a clear description of the question or issue. Do not include passwords, payment details, or sensitive personal information in email.

Future support structure

Built to support each game as the studio grows.

Each current project has a public support route that can be used in App Store Connect when that product is ready for submission.
